# 股票技术指标计算库 stockTechnicalIndicators.py

## 功能描述
stockTechnicalIndicators.py 是一个技术指标计算库，主要用于金融市场（尤其是股票、期货等）的技术分析。它将常见的金融指标计算函数进行了封装，方便用户快速计算各种技术指标。

主要功能分析
1. 代码结构分层
代码采用了清晰的分层设计：

第0层：核心工具函数
这一层提供了最基础的数学运算和数据处理函数，是所有上层函数的基础：

基础运算：RD()（四舍五入）、ABS()（绝对值）、MAX()/MIN()（最大/最小值）

统计函数：MA()（移动平均）、SUM()（求和）、STD()（标准差）

数据处理：REF()（引用之前的数据）、DIFF()（差分）、IF()（条件判断）

极值函数：HHV()（周期内最高值）、LLV()（周期内最低值）

高级统计：AVEDEV()（平均绝对偏差）、SLOPE()（计算斜率）

第1层：逻辑判断函数
基于核心工具函数，实现了技术分析中常用的逻辑判断：

COUNT()：统计满足条件的次数

EVERY()：判断是否一直满足条件

EXIST()：判断是否存在满足条件的情况

CROSS()：判断金叉/死叉（上穿/下穿）

BARSLAST()：计算上一次满足条件到现在的周期数

第2层：技术指标函数
这是代码的核心部分，实现了30多个常用技术指标：

趋势指标：MACD、DMA、TRIX、EXPMA、BBI

摆动指标：KDJ、RSI、WR（威廉指标）、CCI、BIAS（乖离率）

能量指标：OBV、VR（成交量变异率）、MFI（资金流量指标）

路径指标：BOLL（布林带）、KTN（肯特纳通道）

其他指标：ATR（平均真实波幅）、DMI（趋向指标）、PSY（心理线）等

2. 依赖库
NumPy：提供底层的数组计算支持

Pandas：利用其滚动窗口(rolling)和指数加权(ewm)功能实现指标计算

3. 设计特点
函数命名：采用中文拼音缩写，与主流股票软件（如通达信、同花顺）的函数命名保持一致，降低了国内用户的学习成本

向量化计算：基于NumPy/Pandas的向量化操作，计算效率高，适合处理大量数据

返回值处理：多数指标函数最后使用RD()进行四舍五入，使结果更简洁易读

4. 使用场景
这个库非常适合：

量化交易策略开发：快速计算各种技术指标作为策略信号

金融数据分析：对股票、期货等金融时间序列进行技术分析

回测系统建设：为回测系统提供指标计算支持

总结
stockTechnicalIndicators.py 是一个功能强大的技术指标计算库，它通过清晰的分层设计，将复杂的指标计算简化为基础函数的组合。该库的设计特点包括：

函数命名与主流软件一致，降低学习成本

向量化计算，提高计算效率

返回值四舍五入，使结果更简洁易读

本库的主要功能包括：

基础工具函数：提供常用的数学运算和数据处理函数

逻辑判断函数：实现技术分析中常用的逻辑判断

技术指标函数：计算30多个常用的技术指标，包括趋势指标、摆动指标、能量指标、路径指标等

stockTechnicalIndicators.py 是一个轻量级但功能完整的技术指标计算库，它通过简洁的分层设计，将复杂的指标计算简化为基础函数的组合，既保证了代码的可读性，又提供了丰富的技术分析功能。
